Actualización de índices

Sube un Excel para regenerar el namespace de Pinecone con su contenido.

Si el namespace no existe, se crea automáticamente con la carga.

Formatos soportados

  • FAQ (estructura contactoupn) — si el Excel tiene columnas PREGUNTA y RESPUESTA, cada fila se indexa como PREGUNTA: ...\nRESPUESTA: ... con la metadata (docId, line, source, blobType) compatible con los índices existentes.
  • Genérico — cualquier otro Excel: cada fila se indexa como Columna: valor | Columna: valor.

El formato se detecta automáticamente y se muestra en el paso de lectura.
